Para llamar cada círculo se utiliza:

En HTML:
  <iframe class="circle-1" src="percent/circle-1.html" frameborder="0"></iframe>
    * donde el src es el circulo de la carpeta percent que se necesita

En CSS para mejor tamaño y recorte:
  .circle-1{
    width: 50vi;
    clip: rect(3vi, 2vi, 20vi, 2vi);
  }
